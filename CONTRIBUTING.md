# Contributing to Unearth

---

# 1. Philosophy

Unearth values:
- Clean code
- Simplicity
- Mobile-first thinking

---

# 2. How To Contribute

1. Fork repository.
2. Create feature branch.
3. Write clear commit messages.
4. Submit Pull Request.
5. Await review.

---

# 3. Coding Standards

- Vanilla JS preferred.
- Avoid heavy dependencies.
- Optimize for performance.
- Maintain PWA compatibility.

---

# 4. Bug Reports

Open GitHub Issue with:
- Device
- Browser
- Steps to reproduce
- Screenshots (if possible)

---

# 5. Feature Requests

Must include:
- Clear use case
- Benefit explanation
- No feature bloat
