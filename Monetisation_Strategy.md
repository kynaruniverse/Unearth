# Monetisation Strategy – Unearth+

---

# 1. When To Introduce Premium

Conditions:
- At least 200 active users
- 20+ logs per engaged user
- Clear retention above 25%

Do NOT monetize before engagement.

---

# 2. Premium Positioning

Not:
"Upgrade for more features."

Instead:
"Unlock deeper clarity."

---

# 3. Premium Features

- Probability % predictions
- Monthly Chaos Report (PDF)
- Time-of-day pattern insights
- Custom themes
- Cloud sync
- Data export

---

# 4. Conversion Trigger

After 20 logs:

Popup:

"Unearth has detected repeating patterns.
Would you like a deeper reading?"

Buttons:
[Show Me]
[Not Yet]

No guilt copy.

---

# 5. Pricing Strategy

Start:
£2.99/month

Offer:
Founding Member Rate
£1.99/month for first 50 users (lifetime price).

This creates urgency without pressure.

---

# 6. Income Projection Model

5,000 users
5% conversion
£3/month

= £750/month

10,000 users
= £1,500/month

Retention > acquisition.

---

# 7. Psychological Anchors

Free = Utility
Paid = Understanding
Brand = Trust
Founder = Transparency
