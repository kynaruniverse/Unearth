# Unearth Roadmap

---

# Phase 0 – Pre-Build (Weeks 1–2)

- Finalize product brief
- Secure domain
- Set up GitHub repo
- Create design system
- Define microcopy voice

Deliverable:
Functional prototype plan.

---

# Phase 1 – V1 Build & Launch (Month 1)

Build:
- Core tracking
- Ranking system
- Stats page
- PWA install
- Offline mode

Launch:
- Soft Reddit launch
- Build-in-public posts
- Collect feedback

Goal:
100–300 active users.

---

# Phase 2 – Stabilization (Months 2–3)

- Fix UX friction
- Improve prediction logic
- Add small delight features
- Build waitlist for premium

Goal:
Retention > 30%

---

# Phase 3 – V2 Premium Launch (Months 4–6)

Build:
- Auth system
- Cloud sync
- Probability scoring
- Monthly report generator
- Stripe integration

Soft Launch Premium:
Offer discount to early users.

Goal:
First 50 paying users.

---

# Phase 4 – Growth Engine (Months 6–12)

- Launch public case study
- Improve onboarding
- Introduce referral system
- Optimize conversion funnel

Goal:
£1,000–£2,000 MRR

---

# Phase 5 – Scale or Refine (Year 2+)

Options:
- Expand features
- Launch family mode
- Expand into behavioral suite
- Maintain as lean income asset
